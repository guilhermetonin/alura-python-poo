# Trilha: Programar em Python com Orientação a Objetos

> A trilha explora os pilares da linguagem Python e o paradigma de orientação a objetos para a estruturação de códigos com a manipulação de APIs e desenvolvimento de aplicações seguindo as melhores práticas.

> Esse repositório tem o foco em consolidar os fundamentos em Python com a orientação a objetos através do desenvolvimento de projetos práticos e o consumo de APIs.
